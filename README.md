그랬던 프로젝트가 지금은 이렇게 발전했습니다.

1. CSV 파일 게시 - Google 캘린더 '가져오기'와 호환됨
2. ICS 파일 호스팅 - 구독해서 쓸 수 있는 '공개 캘린더'
3. 패키지 배포 - 생성된 ICS, CSV, JSON 파일을 제공함
4. 패키지 배포 - 자바스크립트 `Date`이 공휴일인지 확인
